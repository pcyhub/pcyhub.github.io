2022-10-25日更新日志

测试版正式推送至GitHub
主要用于测试各功能是否可用，及上传GitHub测试

遇到的问题

Spawn failed

出现的问题：未成功连接至GitHub
解决方法：

ssh -T git@github.com
检查是否可连接至Github
若出现
git@github.com: Permission denied (publickey).
原因：
密钥未添加到GitHub
添加重连即可

推送方法

hexo clean && hexo g && hexo deploy

博客生成，页面生成

博客 hexo new 名字
页面 hexo new page 名字
